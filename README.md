# Gravity system
Данная программа моделирует гравитационную систему N материальных точек с произвольными массами, начальными координатами и начальными скоростями. Для исключения удаления материальных точек на бесконечность от центра экрана созданы барьеры за пределами видимости.

Управление:  
- Up - увеличить интервал рассчёта
- Down - уменьшить интервал рассчёта
- LCtrl / RCtrl - менять выбор тела
- a/w/s/d - управление ускорением тела (работает при одновременном нажатии одной кнопки)
- Esc - завершить моделирование
- При желании можно изменять параметры системы при запуске в теле программы (к сожалению, пока нет интерфейса для управления этими параметрами)

graphics.py - используемая готовая библиотека для графического отображения результатов моделирования
